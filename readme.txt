NMR Optimizer written in AMPL for use with MOSEK solver on NEOS.

Code based on thesis by Maryam Khalaji, MSc, CS, 2013, Rapid Re-optimization of Prostate Intensity-Modulated Radiation Therapy Using Regularized Linear Programming

Note: the files are written in AMPL, not Python. They're simply given a .py extension to allow code highlighting.
